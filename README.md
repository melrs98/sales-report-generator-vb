# Sales Report Generator (VB.NET)

## Description 
Built a reporting tool in VB.NET that formats and presents sales data into structured reports, demonstrating the ability to transform raw data into clear and actionable information.

## Technologies
- VB.NET
- Windows Forms
- PrintDocument

## Features
- Generate structured sales reports
- Format data with headers and timestamps
- Display information in a readable layout

## What I Learned
- How to format and present data for reporting
- How to use PrintDocument in VB.NET
- How to transform raw data into useful output
